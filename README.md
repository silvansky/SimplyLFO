# SimplyLFO

<img src="icon.png" width="128" alt="SimplyLFO icon">

macOS app that generates LFO signals and sends them as MIDI CC messages.

## Features

- Multiple independent LFOs
- 10 waveforms: sine, triangle, square, sawtooth, reverse sawtooth, sample & hold, noise, exponential, stepped, smooth random
- Adjustable rate (0.01-20 Hz) and depth
- Configurable MIDI CC per LFO
- Real-time output visualization
- Any MIDI destination

## Requirements

- macOS 13+
- Xcode 15+
