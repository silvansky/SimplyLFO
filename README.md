# SimplyLFO

macOS app that generates LFO signals and sends them as MIDI CC messages.

## Features

- Multiple independent LFOs
- Waveforms: sine, triangle, square, sawtooth
- Adjustable rate (0.1-20 Hz) and depth
- Configurable MIDI CC per LFO
- Real-time output visualization
- Any MIDI destination

## Requirements

- macOS 13+
- Xcode 15+
